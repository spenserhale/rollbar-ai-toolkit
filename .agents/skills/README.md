# Skills

Repeatable agent workflows for this project. Each subdirectory contains a `SKILL.md` with steps, references, and any helper scripts.

Add a skill when a task has well-defined inputs, steps, and expected outputs that an agent would run repeatedly. One-off explanations belong in `docs/` as policy, not here.

## Available skills

- `add-api-operation/` — Add a new Rollbar API operation end-to-end (SDK → CLI → MCP)
