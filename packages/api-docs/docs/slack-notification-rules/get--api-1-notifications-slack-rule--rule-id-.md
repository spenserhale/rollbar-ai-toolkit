# GET /api/1/notifications/slack/rule/{rule_id}

Operation

Get a notification rule by ID

## Parameters

### Path Parameters

| Name      | Type      | Required | Description              |
| --------- | --------- | -------- | ------------------------ |
| `rule_id` | `integer` | Yes      | The notification rule ID |

## Responses

### 200 — Success

**Content-Type:** `application/json; charset=utf-8`

```json
{
  "err": 0,
  "result": null
}
```
