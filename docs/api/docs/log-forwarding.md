# Log Ingestion

Forward logs to Rollbar

In this section, you'll find log forwarding solutions to send errors from your existing logs into Rollbar.

Rollbar SDKs are the preferred way to get data into Rollbar, but sometimes it's easier to get started by forwarding data that's already being written to your logs. We hope that forwarding your existing logs to Rollbar will give your organization a taste of what Rollbar can do and motivate you to do a deeper, SDK-based integration directly into your code in the future.
