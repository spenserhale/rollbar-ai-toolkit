# Rollbar & GDPR

<p>Rollbar is compliant with GDPR and can serve as a data processor for customers who control data from individuals in the EU.  We provide the following measures to enable our customers to maintain GDPR compliance:</p>
<h3>Data Processing Agreement</h3>
<p>Rollbar provides a data processing agreement (DPA) which can be viewed and accepted by going to <strong>Account Settings -&gt; Security -&gt; Data Processing Agreement</strong>.</p>
<h3>Data Retention</h3>
<p>To ensure that you retain your customer's identifiable data no longer than necessary, we've added the option in paid Rollbar accounts to customize your data retention period. You can update this by going to <strong>Account Settings -&gt; Security -&gt; Data Retention</strong>.</p>
<h3>Your Rights</h3>
<p>Subject to applicable law, you have the following rights concerning your Personal Data:</p>
<ul>
<li>Right to erasure</li>
<li>Right to be informed</li>
<li>Right to rectification</li>
<li>Right to restrict processing</li>
<li>Right to data portability</li>
<li>Right to object</li>
<li>Right of access</li>
</ul>
<h3>Data Subprocessor Explanation</h3>
<p>If you are required to list Rollbar as a data subprocessor, the following information can be provided:</p>
<ul>
<li>Subprocessor: <a href="https://rollbar.com">Rollbar</a>
<li>Purpose: Error monitoring </li>
<li>Location: USA </li>
</ul>
