# Directory

Rollbar offers robust support for various languages and frameworks to ensure seamless integration with our platform. Our commitment to providing support is reflected in our defined support levels, ensuring that you can leverage Rollbar's monitoring and error-tracking capabilities with ease and confidence regardless of your development environment.

# Levels of Support Provided

Within our SDK Directory, you'll find a comprehensive breakdown of our support levels for each language and framework.

<table>
<thead>
  <tr>
   <td width="150px">Support Level </td>
   <td>Description </td>
  </tr>
</thead>
<tbody>

  <tr>
   <td>1. Flagship   </td>
   <td>Rollbar's flagship languages and frameworks. When you use Rollbar with one of these languages and frameworks, it's easy to set up, all of the core platform features work, and if you run into a problem, we'll help you solve the issue. These are our most important languages and frameworks.</td>
  </tr>
  <tr>
   <td>2. Supported   </td>
   <td>Supported languages and frameworks. When you use Rollbar with one of these languages and frameworks, it's easy to set up and if you run into a problem, we'll help you solve the issue. </td>
  </tr>
  <tr>
   <td>3. Community</td>
   <td>Not committed to being supported by Rollbar. Language or framework-specific functionality, like an SDK for a language, is provided and supported by the community only. Rollbar may sometimes contribute to the community on a non-committed basis.</td>
  </tr>
  <tr>
   <td>4. Not Supported</td>
   <td>No active community contribution and not supported by Rollbar. These SDK's are open for you to join and contribute to move this into an active community supported SDK.</td>
