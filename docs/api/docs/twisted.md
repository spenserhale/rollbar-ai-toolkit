# Twisted

How to setup pyrollbar with Twisted | Support Level: Community

### Twisted

Please make sure to install twisted and treq first.

```
pip install twisted
pip install treq
```

Check out the [Twisted example](https://github.com/rollbar/pyrollbar/tree/master/rollbar/examples/twisted).
