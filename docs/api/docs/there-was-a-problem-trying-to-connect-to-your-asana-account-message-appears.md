# "There was a problem trying to connect to your Asana account." message appears

<p>This warning message typically appears because the Rollbar app is having problems accessing the Asana API due to an outage or a timeout.</p>
<p>This happens intermittently and is a known issue in the Asana API.  Asana's platform team are working to improve performance and stability of the API to reduce future occurrences.</p>
<p>If you encounter this error, please wait a couple of minutes and try to repeat your action again.</p>
<p>If the error persists, please notify Asana via <a href="https://asana.com/support"><a href="https://asana.com/support">https://asana.com/support</a></a> as well as Rollbar via support@rollbar.com or the in-app chat.</p>
