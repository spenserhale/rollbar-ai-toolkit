# Enable GitHub Organization Access for Rollbar

Sometimes the GitHub repo you want isn't showing up in the dropdown in our UI. If typing the repo name in and saving the settings doesn't work, it may be because you haven't granted permission to the Rollbar application on Github. Go to your personal settings on GitHub, then go to Applications and navigate to the Authorized OAuth Apps. There you should see Rollbar, click on the link and you should be able to grant permission to the organization.
