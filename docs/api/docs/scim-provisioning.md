# SCIM - Users and Teams Provisioning

Rollbar account owners can configure [SCIM](https://en.wikipedia.org/wiki/System_for_Cross-domain_Identity_Management) to provision users and teams in Rollbar.

To start you SCIM setup you will need create an Account Access Token with a SCIM scope for validation with your SCIM provider.  Account access tokens can be found in Settings ->  Account Access Tokens.

The following SCIM providers have been tested and have specific instructions:

* [Okta](/docs/okta#section-step-2-enable-provisioning-of-rollbar-teams-and-users)
* [OneLogin](/docs/onelogin#section-enable-provisioning)
* [Google](https://support.google.com/a/answer/7648945?hl=en\&ref_topic=7662553)
* [Azure](https://docs.microsoft.com/en-us/azure/active-directory/saas-apps/rollbar-provisioning-tutorial)
