# Zenduty

Connecting Rollbar to Zenduty

Rollbar can create Incidents in Zenduty. For instructions on how to configure notifications to Zenduty see the [Zenduty Rollbar Integration Setup Guide ](https://docs.zenduty.com/docs/rollbar).

## What does Zenduty offer Rollbar users?

Zenduty sends new Rollbar alerts to the right team and notifies them based on on-call schedules via email, text messages(SMS), phone calls(Voice), Slack, Microsoft Teams and iOS & Android push notifications, and escalates alerts until the alert is acknowledged or closed.
