# Email Notifications

Email notifications are set up by default on every new Rollbar project. You can configure Email notifications just like other Rollbar integration channels.

## Email Rate Limit

Email notifications are limited to 100 per day, per recipient email address (typically, a user), per Rollbar account. To request a higher limit, please contact <support@rollbar.com>.
