# Flask

How to setup pyrollbar with Flask | Support Level: Supported

### Flask

Please make sure to install `Flask` and `blinker` dependencies first.

```bash
pip install Flask
pip install blinker
```

Check out the [Flask examples](https://github.com/rollbar/pyrollbar/tree/master/rollbar/examples/flask) or [rollbar-flask-example](https://github.com/rollbar/rollbar-flask-example).
